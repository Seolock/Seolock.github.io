# 오로라 아래에서 쓰는 첫 번째 글

> 밤하늘은 언제나 우리에게 더 넓은 세계를 상기시켜준다.

안녕하세요, Aurora 블로그의 첫 번째 포스트입니다.

이 블로그는 생각, 연구, 그리고 일상의 기록을 담는 공간입니다. 오로라가 밤하늘을 수놓듯, 글 하나하나가 모여 의미 있는 빛을 만들었으면 좋겠습니다.

## 블로그를 시작하며

글을 쓴다는 것은 생각을 정리하는 가장 오래된 방법 중 하나입니다. 말로 표현하기 어려운 것들이 문장이 되는 순간, 비로소 실체를 갖게 됩니다.

이 블로그에서는 다음과 같은 주제들을 다룰 예정입니다:

- 연구 과정에서 얻은 통찰
- 책과 논문을 읽으며 든 생각
- 기술과 인문학의 교차점
- 일상의 소소한 발견들

## 마크다운으로 쓰는 글

이 블로그는 마크다운 형식을 지원합니다. 즉, `posts/` 폴더에 `.md` 파일을 추가하면 그대로 렌더링됩니다.

```markdown
# 제목

**굵은 글씨**, *이탤릭*, `코드`

> 인용문

- 목록 항목
```

코드 블록도 예쁘게 표시됩니다:

```python
def aurora():
    """밤하늘에 오로라를 그리다"""
    colors = ["green", "teal", "purple"]
    for color in colors:
        paint_sky(color, alpha=0.4)
```
